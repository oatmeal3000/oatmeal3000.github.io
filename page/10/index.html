﻿﻿﻿奔跑吧，颜色!-Color Run


一款休闲游戏，你需要不停地向前奔跑，永远不要忘了你手中的颜色！使用相应的颜色将迎面而来的物体消除即可，游戏十分需要反应速度和应变能力，你能够在短时间内灵活切换各种颜色吗？
操作方法：

Color Run 具备基本的跑酷游戏类特色——不死不停。

1、迎面而来的障碍，有且只有一种，但是千变万化——由各种颜色组成的图案。

2、你必须在撞击图案前，点击屏幕下方相应色块，将图案由外至内的颜色『融化』。

3、如果最外层有多种颜色，也可以 Multi-touch 多个相应色块，用最快时间解决~

游戏特点：

扁平化设计

还没开始玩，你就能感受到它的设计的独特之处，扁平化，颜色渐变，其实我觉得它非常适合 iOS 7 ，当然，在安卓上也不差。总之，它很特别，从外观开始就很特别。

横版跑酷

游戏开始后主角会不断奔跑，对面碰撞过来的是各种由多种颜色组成的图像，这些图像元素也都是非常扁平化的设计。

颜色消除

它的玩法就像劲舞团和劲乐团那样，对面碰撞过来的是什么颜色的图像，你就得点击底栏上的相应色块来响应并消除图像中的颜色，避免让图像碰到自己，如果点错色块，游戏也会结束。

小鸟飞飞飞-Flappy Bird

《flappy bird》是由来自越南的独立游戏开发者Dong Nguyen所开发的作品，游戏中玩家必须控制一只小鸟，跨越由各种不同长度水管所组成的障碍。
2014年2月，《Flappy Bird》被开发者本人从苹果及谷歌应用商店撤下。2014年8月份正式回归APP STORE，正式加入Flappy迷们期待已久的多人对战模式，作者阮哈东还表示不排除会带来“更多的惊喜”。

操作方法

游戏玩法：在游戏开始后，点击屏幕，要记住是保持一定的规律的仔细观察点击屏幕，不要让小鸟掉下来。
游戏过程：让小鸟不要撞到管子，每穿过一个管子的缝隙得1分。

游戏技巧

在《FlappyBird》这款游戏中，玩家只需要用一根手指来操控，点击触摸屏幕，小鸟就会往上飞，不断的点击就会不断的往高处飞。放松手指，则会快速下降。所以玩家要控制小鸟一直向前飞行，然后注意躲避途中高低不平的管子。
1、在游戏开始后，点击屏幕，要记住是有间歇的点击屏幕，不要让小鸟掉下来。
2、尽量保持平和的心情，点的时候不要下手太重，尽量注视着小鸟。
3、游戏的得分是，小鸟安全穿过一个柱子且不撞上就是1分。当然撞上就直接挂掉，只有一条命。
