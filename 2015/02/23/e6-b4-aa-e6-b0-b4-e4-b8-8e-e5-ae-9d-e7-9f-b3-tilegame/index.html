洪水与宝石原始版 FloodAndGem(Original)

平板电脑

这个网页游戏是 Kevin Reid 的作品, 是 2010 年春季他使用 Visual Basic 做的课程项目。 后来他觉得这个设计挺有意思，应该让更多的人知道， 所以把它移植了一个 Javascript 版本。

概念是你处于一个三维的方块世界， 可以捡起物体还有推箱子，但是不能跳跃和攀爬。 你的目标是要到达五星标记的出口。障碍则是会因为你的举动而泛滥的洪水。用方向键来控制移动，如果是触屏设备，可以在右下角点 Yes 确认使用屏幕虚拟方向键。

尽管游戏的风格也是立体方块，但是 Kevin 在完成这款游戏前还没有听说过 Minecraft 。
