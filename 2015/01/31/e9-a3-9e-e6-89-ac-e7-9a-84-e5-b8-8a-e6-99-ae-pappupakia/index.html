飞扬的帊普 PappuPakia

平板电脑 
这个游戏是为 2012 年为 Github Game Off 制作的。游戏中你是 Pappu, 前进的过程中，你会遇到各种障碍物还有你的敌人 Pakia, 不要撞到他们。 有三种类型的 Pakia, 悲伤的会拉你， 欢乐的会推你， 生气的会消灭你。 和他们要保持安全距离。  

还可以收集些东西！ 黄色的硬币值 50 块， 粉红色的值 100 块， 红色的有 500 块。 星星可以让你保持一段时间的无敌, 浆果也很厉害，你可以试一试。
