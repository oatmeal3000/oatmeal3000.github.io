投稿

方锥网接受科学、科幻及评论、STEM 教育以及未来主题文章投稿，文字形式的作品会通过方锥网的自媒体平台微信公众号 Fundream 发布，其他形式的创作如漫画、HTML5 益智游戏以及自制教育类视频会通过方锥网网站发布。

创作与编剧

注册帐号后请到您的后台发布您的创作。 要求围绕科学主题，立场客观，内容有趣味。 漫画作品请提交宽度为 600px 的背景为透明的 PNG 格式图片。使用由手绘板制作的图片，请保证画面干净清楚。如审核通过会酌情付与一定稿费。也可直接发邮件至方锥网的邮箱 service@fundream.cn。

程序开发者

方锥网关注益智科普主题的游戏， 同时为开发者提供获取劳动收益的平台，也为商户提供了通过游戏推广促销的方式。

开发者提交身份证明资料后，可以立即上传自己的 HTML5 游戏，建议采用 Cocos2d-JS 或 DCloud HBuilder 开发。 可以选取下面列出的两种方式中的一种获取收益。 游戏的内容必须为对青少年完全绿色的益智或者科普主题，否则不予发布。

1. 在游戏开始页面添加广告位

每次广告点击可以为开发者赢得 0.08 ~ 0.1 元的广告收益。方锥网建议游戏页面广告大小采用尺寸为 32050 或 72890。广告图片由商户提供，开发者只要在程序中添加一个接口允许广告显示就可以了。

首先在网页程序中引入用到的脚本：

<script src=”http://fundream.cn/FundreamAuth/jquery.min.js“ type=”text/javascript”></script>

<script src=”http://fundream.cn/FundreamAuth/json2.js“ type=”text/javascript”></script>

<script src=”http://fundream.cn/FundreamAuth/getad.js“ type=”text/javascript”></script>
然后在游戏开始页面任何您想放广告的地方，加一个 id 为 adArea 的 <div>，调用一下 placeAd 函数，把您在方锥网注册的邮件地址作为参数传进去，例如：
<script type=”text/javascript”> placeAd(‘service@fundream.cn’); </script>

<div id=”adArea” style=”text-align:center”></div>


2. 使游戏可以被商户定制

对于单关卡的游戏可以采用这种方式。 对于可以用于商户定制的游戏， 页面下方可由商户输入自己的网站链接， 商户付费后可获得包含 PIN 码的游戏地址， 商户可以自行发布这一链接， 玩家玩完游戏后即可跳转到商户指定的网站页面，并且携带游戏结果给商户。 商户可以根据游戏结果给用户相应的奖励以达到推广自己的目的。

我们为开发者提供了 Javascript SDK 以方便游戏可被商户定制。

首先在网页程序中引入用到的脚本：

<script src=”http://fundream.cn/FundreamAuth/jquery.min.js“ type=”text/javascript”></script>

<script src=”http://fundream.cn/FundreamAuth/json2.js“ type=”text/javascript”></script>

<script src=”http://fundream.cn/FundreamAuth/geturl.js“ type=”text/javascript”></script>
然后在游戏结束的时候调用接口，把您注册方锥网时用的邮箱，和用户的得分作为参数传给函数就可以了，是不是很简单！
redirect2url(‘service@fundream.cn’, 567);
